# Odenwald
Odenwald is a tool which exchange data from points to destination. It is inspired from Collectd(https://github.com/collectd/collectd) and CollectdWin (https://github.com/bloomberg/collectdwin), and support:
- OPC UA
- OPC DA (opc classic)
- InfluxDB
- Kafka
- Console
